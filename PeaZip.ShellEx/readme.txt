To generate a PeaZip app package with W11 mini-context menu support:

1) Place a PeaZip portable package in the (PeaZip repository root)/input directory
Note: place directory (without subfolder)

2) Open (PeaZip repository root)/PeaZip.ShellEx/PeaZip.ShellEx.sln with Visual Studio 2022 or later.

3) Following the instructions https://learn.microsoft.com/en-us/windows/msix/package/packaging-uwp-apps#generate-an-app-package

In this project, the application project is PeaZip.Package
